Functions

Functions let us break up our program into parts that can be run as many times as we want!

  • read_order
  • choose_item
  • locate_aisle
  • navigate_to_aisle
  • find_item
  • grab_item

Each of these can be broken down into more functions too.

Functions (cont.)

This makes our code way more readable, and helps us repeat ourselves less.

Simple Function Example

First we define a function. Then we can call it whenever we want.


                        def do_stuff():
                            print("OOh I'm in a function called do_stuff")
                            print("I can do anything here!")
                        
                        do_stuff() # call the function so that it actually runs!
                        do_stuff() # do it a second time

More Functions

Functions can do more than that example. They can

  • take in values to work with
  • return something when done

Robot Functions

The robot function navigate_to_aisle has to be told what aisle we mean each time we call it.


                        def navigate_to_aisle(aisle_num):
                            print("I am going to asile #" + str(aisle_num))
                            # robot code here...
                        
                        # Later on in the program we can call it with any number for the aisle!
                        navigate_to_aisle(3)
                        navigate_to_aisle(1)
                        navigate_to_aisle(100)

Arguments/Parameters

Functions can receive arguments (also known as parameters) that can be used inside the function.

The function doesn't know what they will hold until they are called with values! 


                        # num1 and num2 are ARGUMENTS
                        def add_together(num1, num2):
                            # Inside here we use them like variables!
                            print(num1 + num2)

Return Values

Functions can return a value to be used after it is called.


                        def add_20(num):
                            answer = num + 20
                            return answer
                        
                        print(add_20(1))
                        print(add_20(100))
                        print(add_20("potato")) # what happens here??
