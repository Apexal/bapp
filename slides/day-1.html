What is Python?

First, what is a programming language?

A programming language is a language in between computer language (101001010) and natural language (Hello, how are you doing?) that allows us to give commands to a computer.

Type in the chat any programming languages you have heard of!

Question

Why can't we program in English?

ANSWER: Natural languages have a level of ambiguity that computers are too dumb to handle! A program written in a programming language has one meaning and one meaning only.

Question

Why don't we program in computer language?

ANSWER: It takes way too long and is really hard to make complex programs with!

Python is one of the most popular programming languages in the world!

It is a good first programming language because it looks a lot like English and generally has less to type!


                        // This is the Go language
                        package main

                        import "fmt"
                        
                        func main() {
                            fmt.Println("hello world")
                        }
                        

                        # This is the same program in Python!
                        print("Hello, world!")
Point out the ugliness of Go and compare it to beautiful Python.

Question

Why are there different programming languages?

Technically, anything you can do in one language can be done in another language. Some languages run faster than others but are harder to code in, some languages only work on some devices or platforms, and some are just preferred by programmers!

Variables

What good are values if we can't keep track of them?

Variables are like labeled boxes we can put values into. They have a name and a value.


                    # Here are some variables:
                    a = 12              # variable a is an integer with value 12
                    b = 13.3            # variable b is a float with value 13.3
                    c = "Hello, world!" # you get the idea!
                    
                    # Variable names can't have spaces, use _ instead!
                    my_name = "Frank"
                    i_love_python = True
