Lists

Keeping track of many things in order.

What is a List?

A list is a collection of things in a particular order.

  • Shopping list
  • Directions
  • Line of people at a store

Python Lists

Python lets us use lists to hold more than one thing.


                        number = 13 # integer variable
                        
                        # Lists!
                        names = ["Alice", "Bob", "Tony", "Susan"] # list of strings
                        my_favorite_numbers = [13, 7, 999] # list of numbers
                        empty = [] # nothing in this list!
                        not_much = [True] # just one boolean here
                        mixing = ["Frank", 20, "NYC"] # different TYPES

Create a List

A list in Python is surround by [ and ]

and each item is separated with a ,

You can put anything in a list!


                        ["Frank", "Nicholas", "Analisa", "Gianna"] # my siblings

List Items

Every item has a position starting at 0.


                        family = ["Frank", "Nicholas", "Analisa", "Gianna"]
                        #            0         1           2          3 

                        # family[0] = "Frank"
                        # family[1] = "Nicholas"
                        # family[2] = "Analisa"
                        # family[3] = "Gianna"

We can get a specific item if we ask for the position!


                        first_child = family[0]
                        second_child = family[1]

                        print(first_child, second_child)

List Length

And the length of the list with len()!


                        num_of_people = len(family)

                        print("There are", num_of_people, "in the family")

Adding Items

Lists would be boring if we couldn't change them!

We can add items easily.


                        names = ["Frank"]

                        # Add items to the END with append
                        names.append("Nicholas")
                        names.append("Analisa")
                        names.append("Gianna")

                        print(names)
Remember that append adds items to the end!

Removing Items

We can remove items easily.


                        names = ["Frank", "Nicholas", "Analisa", "Gianna"]
                        
                        names.remove("Frank")
                        names.remove("Gianna")
                        
                        print(names)

                        # OR if you know where the thing to remove is

                        del names[0] # remove the first item

                        print(names)

Going Through a List

How do we see everything in a list?

Looping through a List

There are two easy ways to go through every item in a list...


                        family = ["Frank", "Nicholas", "Analisa", "Gianna"]

                        # FOR IN LOOP
                        for name in family:
                            print(name)
                        
                        # OR
                        for i in range(len(family)):
                            print(family[i])

Is something in List?


                        family = ["Frank", "Nicholas", "Analisa", "Gianna"]

                        if "Frank" in family:
                            print("Frank is part of the family")
                        
                        if "Rand al'Thor" in family:
                            print("Rand is in the family!")
                        
                        nick = "Nick" in family # This will be either True or False
