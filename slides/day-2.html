Variables

Remember, variables are names for values.


                        var1 = 130 # int variable
                        var2 = 1.1 # float variable
                        var3 = 231 + 13 * 10 # int variable

You use their name in place of a value.


                        print("I.M. Boring")
                        # The above is the same as the below!
                        name = "I.M. Boring"
                        print(name)

What can you name a variable?

  • Can't start with a number
  • Can't have spaces
  • Can't have punctuation besides _
  • Capitalization matters!

                        # Allowed variable names!
                        my_cool_var = "woo hoo"
                        MY_cool_var = "this is a different variable"
                        person13 = "Robyn Banks"
                        WOW_WOW_WOW = 1 # a bad name though...

                        # NOT ALLOWED variables names
                        1movie = "Star Wars"
                        eggs and ham = True
Keep variable names lowercase and separate words with _
The naming rules cannot be broken. Python will not run! The suggestion at the bottom is called a "best practice". Python doesn't care if you follow it, but programmers generally follow that rule so you should too in order to stary consistent!

Controlling the Flow of your Code

Doing the same thing over and over again

Loops

It's common to have to do something many times in a row.

You could copy-paste... why not?

Sometimes we know how many times we want to repeat...

Sometimes we DON'T!

For Loops

When you want to repeat sometime for a certain number of times:



                        # 1) What will this do?
                        for i in range(100):
                            print("Woo hoo!")
                        
                        # 2) What will this do?
                        for pepsi_bottle in range(15): # I was thirsty when I typed this!
                            print("Loop number ", pepsi_bottle) 

                        # 3) Or this?
                        for num in range(10, 21):
                            print("I like the number ", num)

i (or whatever you call it) will hold the number you are up to each time you loop!

While Loops

When want to keep looping while something is true:


                        # 1) What will this do?
                        number = 1
                        while number < 10:
                            print("We're up to", number)
                            number += 1

                        # 2) Or this?
                        while True:
                            print("Uh oh...")
                        
                        # 3) Or this?
                        while False:
                            destroy_the_world()
                            blow_up_moon()
                            print("The End")

Be careful not to cause an infinite loop!

We must go deeper

Loops inside loops inside loops...


                        # Outer loop!
                        while True:
                            year = int(input("Enter a year: "))

                            # Inner loop!
                            for i in range(10):
                                print(i, " year(s) before that it was", year - i)
                    

                        # Multiplication table!
                        for num1 in range(1, 13):
                            for num2 in range(1, 13):
                                print(num1, "times", num2, "equals", num1 * num2)

These are called nested loops.
