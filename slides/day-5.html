Dictionaries

Dictionaries match a key to a value, like a real dictionary!


                        favorite_foods = {
                            # key: value
                            "Frank": "ravioli",
                            "Nicholas": "everything"
                        }

Dictionaries


                        favorite_foods = {
                            # key: value
                            "Frank": "ravioli",
                            "Nicholas": "everything"
                        }

                        # Add or change things!
                        favorite_foods["Sarah"] = "sushi"
                        favorite_foods["Dave"] = "dumplings"

Dictionaries
