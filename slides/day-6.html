Saving and Loading

with files

Why saving and loading?

Our programs can't remember anything after they run!

Imagine a video game where every time you go to play you have to start all over again.

Where do we save things?

Programs can save things in a lot of places. We are going to focus on text files.

Text Files

A text file is like Google Docs but ONLY text with nothing fancy at all:

stuff.txt

                        im a boring text file
                        
                        no color
                        no different sizes
                        no pictures :(

Reading a Text File


                        # Open up the file named stuff.txt
                        with open("stuff.txt") as f:
                            # Read in all of the lines into a list
                            lines = f.readlines()
                            # Print out the lines
                            print(lines)

What is \n?

\n means there is a new line!


                        print("hello\nworld\n\nempty lines!")

Prints


                        hello
                        world
                        
                        empty lines!

Getting Rid of \n

We don't want \n at the end of every line. Use strip() to get rid of spaces and \n. 


                        with open("stuff.txt") as f:
                            for line in f.readlines():
                                print(line.strip())

Reading Todos

What if we added this to the top of our TODO program?


                        with open("todos.txt") as f:
                            for line in f.readlines():
                                todos.append(line.strip())

Writing a Text File

Just add "w" to write to the file! 


                        with open("stuff.txt", "w") as f:
                            f.write("Hello world!")
NOTE: This will get rid of what was in the file!

Saving Todos

Let's add a save command to our TODO program.


                        with open("todos.txt", "w") as f:
                            for todo in todos:
                                f.write(todo + "\n")
